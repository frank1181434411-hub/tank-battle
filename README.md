# Tank Battle

Tank Battle 是一个使用 C++17 和 SFML 3.1.0 开发的复古坦克大战课程设计项目。当前仓库已经具备基础战斗原型，后续计划继续补全地图、AI、联机、编辑器、音效动画和可选强化学习 AI。

## 当前状态

- 已完成：窗口创建和主循环、玩家 WASD 移动、玩家射击、炮弹飞行与越界销毁、玩家和敌方/基地碰撞伤害、血包、弹药强化包、轻型和重型敌方坦克、最小胜负判定。
- 部分完成：敌人基础移动和自动射击、固定场景中的敌人与道具生成、基地生命与受击逻辑。
- 已建立接口或空文件：地图模块、资源管理模块、测试文件、配置文件。
- 计划功能：地图加载、地形碰撞、多难度 AI、地图编辑器、局域网联机、HUD/菜单、音效、动画、可选强化学习 AI。

## 构建与运行

在 MSYS2 UCRT64 终端中执行：

```bash
cd /d/CPP/tank-battle
cmake --preset debug
cmake --build --preset debug
./build/debug/bin/TankBattle.exe
```
